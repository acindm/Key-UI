# Loading 加载

网页处于加载过程时显示的动画。

## 使用场景

加载过程中显示的动画。或在等待加载的位置提供一个占位图。

## 常见加载图标

最基础的加载图标，直接在标签内设置大小与颜色（图标默认颜色为主题色）。
